# Cofobol

Site institucional do COFOBOL desenvolvido com Next.js, React e TypeScript.

## Stack

- Next.js 16
- React 19
- TypeScript
- Firebase
- Vercel

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run lint
```

## Deploy

Produção em:

`https://cofobol.vercel.app`
